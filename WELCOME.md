## 🎯 Features

- User authentication (signup/login)
- Train search (mock API)
- Create ride intents
- Smart ride matching using Haversine formula
- Send/receive pooling requests
- Real-time chat
- User profiles with ratings
- Responsive UI

## 🔧 Tech Stack

**Backend**: Flask + PostgreSQL + SQLAlchemy
**Frontend**: React + Axios + React Router
**Auth**: JWT
